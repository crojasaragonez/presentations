Text Tools

  • cat (prints files)
  • tail (prints last N lines of a file)
  • head (prints firts N lines of a file)
  • less (reads the content of a file)
  • sed (search and replaces text within files)
  • grep (finds text within files)
  • awk (helps formatting terminal output)
  • locate (finds any file in hard disk in seconds)

Piping

Transfer output of a tool to some other destination or tool


						echo 'Hello world' | sed 's/world/World/i' #Hello World
						echo 'Hello world' | sed 's/world/World/i' | base64 #SGVsbG8gV29ybGQK
						echo 'SGVsbG8gV29ybGQK' | base64 --decode #Hello World
