Containerization


The "new" kid in the block, since ~2014

What is it?

It allow us to virtualize applications no matter the techbology or if we have installed a single programming language in our current machine.

Advantages


  • Containers are normally very light weight (MB and not GB).
  • We can run entire and very complex apps in minutes and not hours.
  • Built to scale.
  • Main cloud providers already support containers (AWS, Azure, GCP).
  • You could deploy hundred or even thousand servers with a single command line when you integrate it with kubernetes.
